Ride Scheduler (PHP 5.6 + SQLite)

- Upload the folder to your web root (e.g. /var/www/html/ride-scheduler).
- Ensure the web server can write to:
  - /db (SQLite file will be created/updated here)
  - /logs (application log)
- First run:
  - If no admin user exists, you will be redirected to create the first admin.
